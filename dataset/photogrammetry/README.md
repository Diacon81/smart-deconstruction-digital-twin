# Photogrammetry

This directory contains the resources associated with the photogrammetric reconstruction stage of the proposed methodology.

The photogrammetric process transforms the UAV image dataset into a three-dimensional representation of the building, providing the geometric basis required for digital twin generation, structural characterization and future robotic simulation.

Future contents may include:

- Camera alignment data.
- Sparse and dense point clouds.
- Mesh reconstruction.
- Textured 3D models.
- Orthophotos.
- BIM-oriented reconstruction outputs.

These resources constitute one of the core components of the smart deconstruction workflow presented in the Master's Thesis.
