# Simulation

This directory contains the resources associated with the simulation stage of the proposed smart deconstruction framework.

The simulation environment is based on CoppeliaSim and is intended to validate inspection strategies, collision avoidance, navigation and future robotic interventions using the digital twin generated from photogrammetric data.

Future contents may include:

- CoppeliaSim scenes.
- Virtual robot models.
- Sensor configurations.
- Navigation and path-planning experiments.
- Collision detection analyses.
- Simulation screenshots and validation outputs.

This stage supports the virtual evaluation of the proposed methodology before its potential implementation in real selective deconstruction scenarios.
