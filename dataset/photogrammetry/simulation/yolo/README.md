# YOLO

This directory contains the computer vision resources associated with the proposed smart deconstruction framework.

The YOLO (You Only Look Once) object detection algorithm is incorporated to support the automatic identification of visible construction elements, materials and potential risk areas from UAV-acquired images.

Future contents may include:

- Image annotations.
- Training and validation datasets.
- Configuration files.
- Trained YOLO models.
- Detection results.
- Inference scripts and performance metrics.

The objective of this module is to improve the semantic characterization of abandoned or unfinished buildings, providing useful information for digital twin generation, selective deconstruction planning and future robotic intervention.
