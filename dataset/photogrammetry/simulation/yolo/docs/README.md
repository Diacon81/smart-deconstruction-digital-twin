# Documentation

This directory contains the main scientific documentation associated with the research project.

Contents

- **Master's Thesis** – Complete academic document describing the theoretical background, methodology, experimental validation, results and conclusions of the proposed framework.

- **Scientific Journal Article** *(currently under preparation)* – Future publication derived from the Master's Thesis, intended to present the proposed methodology to the international research community.

The documentation available in this directory complements the technical material contained in the repository and provides the scientific context required to understand, reproduce and extend the proposed smart deconstruction digital twin framework.
