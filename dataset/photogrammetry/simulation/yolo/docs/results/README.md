# Results

This directory contains the results obtained during the experimental validation of the proposed smart deconstruction framework.

Current and future contents may include:

- Photogrammetric reconstruction outputs.
- Digital twin visualizations.
- YOLO detection examples.
- Performance metrics.
- Simulation screenshots.
- Risk assessment results.
- Validation figures and comparative analyses.

These resources support the evaluation of the proposed methodology and provide complementary material to the results discussed in the Master's Thesis.
